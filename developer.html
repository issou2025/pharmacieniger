Passionné par les technologies modernes et l'innovation numérique au Niger, je conçois des solutions logicielles sur mesure, performantes et accessibles pour relever les défis locaux.

À travers **Pharma Niamey**, mon objectif a été de concevoir une application ultra-rapide, fonctionnelle 100% hors-ligne (PWA) et libre de droits, pour aider les habitants de Niamey à trouver rapidement l'aide médicale urgente dont ils ont besoin, de jour comme de nuit.

🛠️ Compétences clés
JavaScript Vanilla (ES6+) HTML5 & CSS3 Premium Python & Scraping Avancé Progressive Web Apps (PWA) Cartographie Leaflet.js Git & CI/CD Actions Optimisation de Performance
💡 Réalisation en Vedette

Pharma Niamey

Open Source

Un localisateur de pharmacies et d'officines de garde fonctionnant sans base de données SQL lourde ni framework JavaScript complexe. L'application extrait dynamiquement et met à jour automatiquement les données deux fois par jour via un script de scraping automatisé par GitHub Actions, le tout servi instantanément depuis GitHub Pages.
